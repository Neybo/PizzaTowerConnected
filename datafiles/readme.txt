PTC is a mod of the original PTO left as it was from the github, we've fixed it up and added more features. 
Press F5 at the titlescreen to open the manual for more info on some bits.

Features:
- A shitton (too many to describe)
- peter griffin peppino now appears in pause menu


Commands:
- /afk - sends you to an afk room
- /t or /minichat - toggles minichat
- /tp - sends a tp request 
- /tpyes - accepts a tp request
- /tpno - declines a tp request
- /list - Shows a list of all other players
- /ping - Returns your ping in milliseconds
